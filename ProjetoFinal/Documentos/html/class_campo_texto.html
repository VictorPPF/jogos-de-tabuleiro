Detailed Description

Classe que representa um campo de texto interativo, permitindo entrada e exibição de texto.

Classe que representa um histórico, onde armazena os dados das partidas.

Esta classe gerencia a entrada de texto, a renderização e o estado do campo de texto na tela.

Esta classe gerencia a entrada e armazenamento de texto.
