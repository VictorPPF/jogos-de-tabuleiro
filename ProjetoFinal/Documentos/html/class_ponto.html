Detailed Description

template<typename T>
class Ponto< T >

Representa um ponto no plano cartesiano.

Esta classe template permite a criação de pontos com diferentes tipos numéricos (double, float, int, unsigned int). Fornece operadores sobrecarregados para operações aritméticas e comparação entre pontos.
